> 添加标签 可以用  
>
> 1. HEAD 
> 2. 甚至可以HEAD^   为前面的提交增加标签, 
>
> 2. master(分支名) 
>
> 3. 哈希值. 
>
> 4. 默认
>
>    

执行 `git push origin [标签名]` 推送标签到远程仓库，注意前面的命令都只涉及本地操作不需要联网，此命令需要联网：

![此处输入图片的描述](6.4_将本地标签推送到远程仓库.assets/document-uid310176labid10349timestamp1553931743455.png)
