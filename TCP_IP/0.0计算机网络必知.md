常用协议对应端口号：

- SSH 22
- FTP 20 和 21
- Telnet 23
- SMTP 25
- TFTP 69
- HTTP 80
- SNMP 161
- **应用层：应用程序通过这一层访问网络，常见 FTP、HTTP、DNS 和 TELNET 协议；**
- **传输层：TCP 协议和 UDP 协议；**
- **网络层：IP 协议，ARP、RARP (ARP的逆向, MAC->IP)协议，ICMP 协议等；**
- **网络接口层：是 TCP/IP 协议的基层，负责数据帧的发送和接收。**

- 物理层（一层）PDU 指数据位（Bit）
- 数据链路层（二层）PDU 指数据帧（Frame）
- 网络层（三层）PDU 指数据包（Packet）
- 传输层（四层）PDU 指数据段（Segment）
- 第五层以上为数据（data）
