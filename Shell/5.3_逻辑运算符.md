#### 逻辑运算符



![5-3-1](5.3_逻辑运算符.assets/document-uid8797labid3895timestamp1554172035321.png)

实例

```bash
#!/bin/bash
a=10
b=20

if [[ $a -lt 100 && $b -gt 100 ]]
then
   echo "return true"
else
   echo "return false"
fi

if [[ $a -lt 100 || $b -gt 100 ]]
then
   echo "return true"
else
   echo "return false"
fi
```

结果

```bash
return false
return true
```
