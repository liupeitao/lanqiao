浮点运算，比如实现求圆的面积和周长。

`expr` 只能用于整数计算，可以使用 `bc` 或者 `awk` 进行浮点数运算。

```bash
#!/bin/bash

radius=2.4

pi=3.14159

girth=$(echo "scale=4; 3.14 * 2 * $radius" | bc)

area=$(echo "scale=4; 3.14 * $radius * $radius" | bc)

echo "girth=$girth"

echo "area=$area"
```

> 以上代码如果想在环境中运行，需要先安装 `bc`。

```bash
sudo apt-get update
sudo apt-get install bc
```
