#### 条件测试

条件测试表达式放在 `[ ]` 中。下列练习中的 `-lt` (less than)表示小于号。

```bash
vim test24.sh
```

输入代码：

```bash
#!/bin/bash

a=5
if [ $a -lt 10 ]
then
    echo "a: $a"
else
    echo 'a>=10'
fi
```

运行代码：

```bash
bash test24.sh
a: 5
```

**双中括号（`[[ ]]`）也用作条件测试（判断）**，后面的实验会详细讲解。

#### 数组元素

在一个 array 结构的上下文中，中括号用来引用数组中每个元素的编号。

```bash
vim test25.sh
```

输入代码：

```bash
#!/bin/bash

arr=(12 22 32)
arr[0]=10
echo ${arr[0]}
```

运行代码：

```bash
bash test25.sh
10
```
