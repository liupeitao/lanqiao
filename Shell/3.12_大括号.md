#### 文件名扩展

复制 t.txt 的内容到 t.back 中

```bash
vim test22.sh
```

输入代码：

```bash
#!/bin/bash

if [ ! -w 't.txt' ];
then
    touch t.txt
fi
echo 'test text' >> t.txt
cp t.{txt,back}
```

运行代码：

```bash
bash test22.sh
```

查看运行结果：

```bash
ls
cat t.txt
cat t.back
```

注意： 在大括号中，不允许有空白，除非这个空白被引用或转义。

#### 代码块

代码块，又被称为内部组，**这个结构事实上创建了一个匿名函数**（一个没有名字的函数）。然而，与“标准”函数不同的是，在其中声明的变量，对于脚本其他部分的代码来说还是可见的。

```bash
vim test23.sh
```

输入代码：

```bash
#!/bin/bash

a=123
{ a=321; }
echo "a = $a"
```

运行代码：

```bash
bash test23.sh
a = 321
```

变量 a 的值被更改了。
