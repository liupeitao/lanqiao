# 更多标记

- `''` 是上次跳转操作光标所在的位置
- `'"` 是上次退出当前文件的位置
- `'[` 是上次编辑（删除修改等）的开始位置
- `']` 是上次编辑（删除修改等）的结束位置
- `'.` 是上次编辑（删除修改等）的最后的操作位置
